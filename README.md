# 🤖 AI Text Summarizer

This is a simple AI-based text summarizer using Groq API.

## 🚀 Features
- Summarize long text instantly
- Clean UI
- Copy summary feature

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- Groq API

## ⚙️ Setup

### 1. Clone the repo
git clone https://github.com/your-username/ai-summarizer.git

### 2. Install backend dependencies
cd backend
npm install

### 3. Add .env file
Create a .env file in backend folder:

GROQ_API_KEY=your_api_key_here

### 4. Run server
node server.js

### 5. Open frontend
Open index.html in browser

## ⚠️ Note
.env file is not included for security reasons.
